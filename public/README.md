# Arquivos Públicos da PontoVet

## Logo

Por favor, adicione a logo da PontoVet nesta pasta com o nome `logo.png`.

A logo deve ter fundo transparente ou branco para melhor visualização no site.

## Galeria

Para adicionar fotos na galeria:
1. Crie uma pasta chamada `gallery` dentro de `public/`
2. Adicione as fotos da clínica (foto1.jpg, foto2.jpg, etc.)
3. Atualize o componente `pages/galeria.vue` para referenciar as imagens reais

Exemplo:
```
public/
  ├── logo.png          (sua logo aqui)
  └── gallery/
      ├── foto1.jpg
      ├── foto2.jpg
      └── foto3.jpg
```
